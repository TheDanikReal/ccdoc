# $closeTicket
Closes a ticket created by the bot using [`$newTicket`](../Channel/newTicket.md).


#### Usage: `$closeTicket[error message(optional)]`
<br/>

```sh
   $sendmessage[This ticket will be closed in 5 seconds.]
   $wait[5s]
   $closeTicket
```

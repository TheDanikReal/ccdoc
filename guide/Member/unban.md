# $unban
Bans a user from the server

#### Usage: 
`$unban[userID;reason]`

If member can be banned by the bot:
<br/>
<discord-messages>
	<discord-message :bot="false" role-color="#ffcc9a" author="Discord Moderator">
		!!exec $unban[898908909879675454;Unban because ban appeal]
	</discord-message>
</discord-messages>

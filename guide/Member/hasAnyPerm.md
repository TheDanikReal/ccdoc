# $hasAnyPerm

Checks if a user or the author has one of the provided permissions

## Usage

```bash
$hasAnyPerm[permission1;permission2;...] or $hasAnyPerm[userID;permission1;permission2;...]
```

## Example
```
$hasAnyPerm[$userID;manageroles;managechannels]
```
